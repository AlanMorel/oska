# Oska

A Discord bot written using Node in TypeScript

Install node, npm, clone this repository, then install the dependencies

```sh
git clone https://github.com/AlanMorel/oska
```

```sh
cd oska
```

```sh
npm install yarn -g
```

```sh
yarn install
```

Create a `.env` file

```sh
TOKEN=
PREFIX=!
LOCALE=en-US
```

Create a folder called `logs` at the root

```sh
mkdir logs
```

Run the bot

```sh
yarn start
```
